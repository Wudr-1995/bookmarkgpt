# BookmarkGPT — AI Chrome Extension

**AI-powered bookmarks that actually make sense.**

Save any webpage with one-click AI summarization. No API key needed — uses Chrome's built-in AI (Gemini Nano).

## ✨ Features

- **🤖 AI Summaries** — Automatically generated by Chrome's built-in AI
- **🏷️ Auto-Tags** — AI suggests relevant tags for each bookmark
- **🔍 Full-Text Search** — Search across all your bookmarks by title, URL, summary, or tags
- **⚡ Keyboard Shortcut** — `Ctrl+Shift+B` (or `Cmd+Shift+B` on Mac) to quick-save
- **🔒 Privacy First** — All processing happens locally in Chrome
- **📱 Cross-Device Sync** — Uses Chrome Storage (sign in to Chrome to sync)
- **💰 Completely Free** — No API key, no server, no cost

## 🚀 Installation

### From Source

1. Download/clone this repository
2. Open Chrome → `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the `bookmarkgpt` folder

## 📖 Usage

### Method 1: Click Extension Icon

1. Navigate to any webpage
2. Click the BookmarkGPT icon in Chrome toolbar
3. Review the AI-generated summary
4. Edit tags if needed
5. Click "Save Bookmark"

### Method 2: Keyboard Shortcut

1. Navigate to any webpage
2. Press `Ctrl+Shift+B` (Mac: `Cmd+Shift+B`)
3. See notification that page is ready
4. Click BookmarkGPT icon to save

### Search Your Bookmarks

1. Click the BookmarkGPT icon
2. Type in the search box
3. Results update instantly as you type

## 🔧 How It Works

BookmarkGPT uses Chrome's built-in AI (Gemini Nano) to generate summaries. This means:

- **No API key required** — Works completely offline
- **No data sent to servers** — Everything processes locally
- **Fast summarization** — Uses Google's on-device AI model

### Browser Requirements

- Chrome 131+ (with Gemini Nano support)
- Or any Chromium-based browser with Prompt API

If your browser doesn't support the Prompt API, BookmarkGPT falls back to extractive summarization from the page title and URL.

## 🎹 Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+B` | Mark current page as "ready to save" |
| `Cmd+Shift+B` (Mac) | Mark current page as "ready to save" |

After pressing the shortcut, click the extension icon to complete the save.

## 📊 Tech Stack

- **Manifest V3** Chrome Extension
- **Chrome Prompt API** (Gemini Nano) for AI
- **Chrome Storage API** for persistence
- Pure HTML/CSS/JS — no build step required

## 🔒 Privacy

- All AI processing happens locally using Chrome's built-in AI
- No external API calls
- Bookmarks stored in your Chrome profile
- No analytics, tracking, or third-party services
